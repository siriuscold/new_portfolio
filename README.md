# Портфолио Danya Antonov

Сайт-портфолио с интерактивными перетаскиваемыми изображениями.

## Особенности

- Перетаскивание изображений мышью или касанием
- Ограничение перемещения границами экрана
- Плавная анимация увеличения при наведении (hover)
- Адаптивный дизайн

## Использование

Просто откройте `index.html` в браузере. Для локального запуска можно использовать:

```bash
python -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
